# copywriting

David has two styles. Pick the style by surface, then apply the core rules.

## pick the style

- **authentic**: casual surfaces where David speaks as himself. tweets, DMs, community posts, GitHub About, personal READMEs.
- **formal**: official surfaces. business emails, sponsor communication, support text, docs for outsiders.
- unsure which fits? ask the user.

## core rules (both styles)

- short sentences
- no emojis
- minimum hype words
- no em dashes
- numbers as digits (spell the word only when it clearly reads better)
- high perceived value, never needy or salesy
- a bit of mystery beats overexplaining
- cut every word that doesn't earn its place

## authentic style

- all lowercase. no capitals anywhere, not even names or sentence starts
- profanity when it makes sense
- reads like a quick personal message, not marketing

## formal style

- standard capitalization and punctuation
- no profanity
- still concise and direct. formal does not mean bloated

## never do this (obvious ai tells)

David hard-rejects these patterns. They instantly read as ai-written:

- **subjectless verb openers**: "made my skills public", "built a new tool". a sentence never starts with a bare verb out of nowhere. always "i made", "just made", "i built".
- **the contrast punchline**: "not a demo, the real files", "this is not a breakthrough, this is a revolution". no human writes like that. state the thing directly instead.
- **label-colon openers**: "quick update: ...", "big news: ...". never a short label plus colon plus the actual text. just write the sentence.

## approved examples (david-picked)

- tweet: "just open-sourced my personal agent skills. the exact files my agents run on. take them."
- github about: "access to david ondrej's personal agent skills"
- casual dm: "yo, just made my agent skills public. steal whatever's useful"
- readme opening: "skills my agents actually use, synced straight from my mac."
- formal email opening: "Hello Marek, I've published the skills repository we discussed. Here is the link:"

why they win: plain statement of what happened, concrete facts, casual and honest, not try-hard, no pitch.
